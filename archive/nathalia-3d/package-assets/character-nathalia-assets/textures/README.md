# Nathal.IA — Texturas

Texturas (albedo/normal/roughness) e materiais auxiliares dos modelos 3D.
Vazio nesta fase.

## Recomendações

- Formato: `.webp` ou `.ktx2` (compressão Basis) para web; `.png` apenas quando
  necessário.
- Resolução: 1024² para rosto/camiseta; 512² para detalhes.
- Camiseta preta com wordmark **jump** (ver paleta em `docs/nathalia/ASSET_GUIDE.md`).
- Embuta as texturas no `.glb` quando possível para simplificar o carregamento.
