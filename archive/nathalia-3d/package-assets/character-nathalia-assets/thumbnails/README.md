# Nathal.IA — Thumbnails

Pré-visualizações 2D (PNG/WebP) de cada pose, usadas como fallback estático e em
catálogos/documentação. Vazio nesta fase.

## Convenção

- `nathalia-<estado>.webp` (ex.: `nathalia-idle.webp`, `nathalia-happy.webp`).
- Fundo transparente, enquadramento de busto, ~256×256.
- Servem como fallback quando o WebGL falha e não há avatar 2D dinâmico.
