# `base/` — Modelos aceitos para refinamento

Candidatos que **passaram no intake** e foram aceitos como ponto de partida para
o refinamento no Blender (Fase 4). Nome canônico: `nathalia_base.glb`.

- Um arquivo entra aqui somente após decisão **"aceitar para refinamento"** no
  [`ASSET_INTAKE_REPORT.md`](../../../docs/nathalia/ASSET_INTAKE_REPORT.md).
- Binários **não versionados** (ADR-010 / D-004).
- Daqui o Blender produz o candidato a master em `../master/`.
